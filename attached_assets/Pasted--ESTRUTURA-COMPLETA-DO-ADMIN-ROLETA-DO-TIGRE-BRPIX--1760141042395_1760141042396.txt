🧩 ESTRUTURA COMPLETA DO ADMIN — ROLETA DO TIGRE / BRPIX

------------------------------------------------------------
📊 1. DASHBOARD PRINCIPAL
------------------------------------------------------------
Painel de visão geral com métricas em tempo real.

- Lucro do Jogo: mostra saldo positivo ou negativo (ex.: R$ -383,50)
- Total Apostado (Jogo): soma geral de todas as apostas realizadas
- Depósitos de Hoje: total de depósitos confirmados no dia
- Prêmios Pagos (Jogo): total de valores pagos aos jogadores
- Total de Usuários: número de usuários cadastrados (com variação percentual)
- Depósitos Confirmados: soma dos depósitos aprovados via BRPIX
- Saldo Total em Contas: valor total armazenado em contas internas
- Saques Pagos: total de saques processados com status de pendente ou concluído

📈 Gráficos incluídos:
- Depósitos (Últimos 7 dias)
- Maiores Saldos de Usuários

------------------------------------------------------------
👥 2. GESTÃO
------------------------------------------------------------

▪️ Usuários
- Lista todos os usuários cadastrados
- Exibe dados: nome, e-mail, saldo, status, data de cadastro
- Ações: Editar, Bloquear, Excluir

▪️ Afiliados
- Lista de afiliados com ID de referência e comissões geradas
- Exibe total de usuários indicados e ganhos acumulados
- Opções: Visualizar relatórios, Gerar link de afiliado, Exportar CSV

▪️ Depósitos
- Histórico de depósitos realizados via BRPIX
- Campos: ID, valor, data, status (Pendente, Confirmado, Falhou)
- Botões: Confirmar Manualmente ou Estornar
- Integração direta com API BRPIX (/transactions)

▪️ Saques
- Lista de solicitações de saque
- Exibe valor, chave Pix, data e status
- Opções: Aprovar (via API BRPIX) ou Rejeitar
- Logs automáticos de cada operação

------------------------------------------------------------
🎮 3. JOGOS
------------------------------------------------------------

▪️ Roleta Principal
- Configuração de probabilidades (%) de cada multiplicador
- Prêmios: 0x, 2x, 5x, 10x, 15x, 20x, 30x, 50x, 100x
- Cada linha: “Prêmio (x)” + campo “Probabilidade (%)” + botão “Salvar”
- Dados armazenados em `roleta_principal`

▪️ Roleta Bônus
- Configuração separada para rodadas bônus
- Multiplicadores: 1x, 2x, 3x, 4x, 5x
- Controle de porcentagem individual
- Armazenado em `roleta_bonus`

------------------------------------------------------------
⚙️ 4. SISTEMA
------------------------------------------------------------

▪️ Logs e Auditorias
- Histórico de ações de usuários e admins
- Campos: ID, tipo, data, IP, resultado

▪️ API BRPIX (Integração)
- Configuração de chaves e endpoints:
  - BRPIX_PUBLIC_KEY
  - BRPIX_SECRET_KEY
  - BRPIX_ENDPOINT
- Testes diretos de requisições (status, depósitos, saques)
- Campo para token administrativo

▪️ Webhooks
- URLs configuráveis para eventos:
  - PIX_CONFIRMED
  - PIX_FAILED
  - WITHDRAW_CONFIRMED
  - WITHDRAW_FAILED

------------------------------------------------------------
💼 5. AUTENTICAÇÃO E PERFIL
------------------------------------------------------------
- Exibe nome do admin (ex.: NextSistemas) e usuários online
- Menu: Meu Perfil, Alterar Senha, Sair (Logout)

------------------------------------------------------------
💬 6. NOTIFICAÇÕES
------------------------------------------------------------
- Alertas dinâmicos:
  - Usuário online
  - Novo depósito confirmado (BRPIX)
  - Saque aprovado
- Configuração em Sistema → Webhooks

🧾 7. RELATÓRIOS E ESTATÍSTICAS
------------------------------------------------------------
- Relatórios exportáveis (CSV e PDF):
  - Usuários
  - Depósitos
  - Saques
  - Jogos
- Filtros por data e valor
- Exportação com nome do admin e data

------------------------------------------------------------
🔐 8. LOGIN E SESSÃO
------------------------------------------------------------
- Tela de login com logo “Roleta do Tigre”
- Campos: E-mail, Senha
- Botão Entrar
- Sessão persistente (jwt / sessionStorage)

------------------------------------------------------------
🔄 9. INTEGRAÇÃO BRPIX (GATEWAY OFICIAL)
------------------------------------------------------------
- BRPIX como gateway principal de transações Pix
- API RESTful:
  - POST /transactions/create
  - GET /transactions/status
  - POST /transactions/refund
  - POST /withdraw/request
- Campos obrigatórios: valor, chave_pix, tipo, referencia, token_admin
- Webhook confirma depósitos e saques automaticamente
